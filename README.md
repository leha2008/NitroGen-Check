## Installation
  1. Install Python packages:
      ```sh
      #Windows:
      Run the setup-windows.bat
      #Unix:
      chmod +x setup-unix.sh && ./setup-unix.sh
      ```
## Usage
  2. Usage Nitro Gen
      ```sh
      #Windows:
      Run the start-windows.bat
      #Unix:
      chmod +x start-unix.sh && ./start-unix.sh
      #Termux:
      chmod +x termux-start.sh && ./termux-start.sh
      ```
